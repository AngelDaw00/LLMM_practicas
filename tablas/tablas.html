Tablas en HTML

Hasta hace cosa de unos 10-15 años aproximadamente, todo lo que era maquetación web se hacía mediante tablas.
Hoy en día, las tablas NO se utilizan para maquetar, sino para insertar cualquier tipo de archivo como pueden ser
audios, vídeos, imágenes, formularios, otras tablas... etc. y son una forma de organizar contenido en
nuestras webs.
Cuando vayamos a crear/utilizar una tabla en nuestras páginas webs, deberemos tener en cuenta las siguientes
propiedades y características:
  • table: sirve para indicar que vamos a utilizar una tabla. Tiene etiqueta de apertura y cierre.
  • tr: es la que va a determinar cuantas filas vamos a tener en nuestra tabla.
  • td: es el elemento que va a determinar cuantas celdas van a ir insertadas en cada una de las
    filas de nuestra tabla.

Propiedades a tener en cuenta a la hora de diseñar una tabla

  • table: sirve para definir que vamos a utilizar una tabla
  • caption:Sirve para poder establecer un título de cabecera de nuestra tabla.
    Podemos alinear este en la parte superior (top) o en la parte inferior (bottom)
  • thead o th: definir la cabecera de una tabla
  • tbody: define el contenido de la tabla
  • tfoot: define el pie de la tabla
  • tr: define la fila para la tabla
  • td: define una celda para una fila de la tabla.

Ejemplo de combinación de celdas en tablas

Para poder combinar celdas en tablas, podemos utilizar las siguientes propiedades:
  • rowspan: se utiliza para indicar el número de columna que ocupa la celda (combina columnas)
  • colspan: se utiliza para indicar el número de filas que ocupa la celda (combina filas)
